## Description

A sample demonstrating serving a VueJS website from a NestJS backend.

## Installation

```bash
$ npm install
$ npm run build --prefix client
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
